UI Test Automation

Example UI automation tests implemented using Java, Selenium Web Driver and TestNG.

The project follows the POM design pattern.

Covered functionality:

The tests demonstrate automation of selected scenarios, including:

- creating and managing physical products
- managing products inside a package (changing order and removing items)

Tech stack
- Java  
- Selenium WebDriver  
- TestNG  
- Maven  

Notes

Tests require access to the target application and valid credentials provided by the system administrator.

Example executions of the tests are shown in the attached recordings.
